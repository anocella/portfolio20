Our Process

This site is intended to be used as a tool for portfolio in the context of modern portfolio theory, or MPT. A major conclusion that comes from MPT is that an ideal portfolio either provides the best return for a given level of risk, or provides the lowest risk for a given level of return. A very easy way to achieve lower risk is to combine multiple assets with similar levels of return, but low correlation to each other, so that the resulting portfolio is more efficient: without much loss of expected return, we can greatly reduce our risk of not achieving that return.

Correlation measures the tendency for two assets to have above- or below-average returns simultaneously; high correlations are therefore bad for the portfolio because there is no diversification when all of your assets do well or poorly at the same time. Adding assets that have a low or negative correlation to an existing portfolio make it more efficient, which we measure with the Sharpe ratio. The Sharpe ratio is defined as the ratio of excess return (portfolio return less return on cash) to portfolio risk (measured as the standard deviation of returns). There is one theoretical "best" portfolio that has the highest Sharpe ratio, and the goal of portfolio optimization is to estimate that portfolio.

We use daily returns for all of our calculations, and we express results in annualized terms where applicable. We annualize return by multiplying the daily value by 252, and we annualize risk by multiplying the daily value by the square root of 252.
